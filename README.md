# SKILLDD Premium App 🛠️

SKILLDD is a mobile-first web app prototype designed for community skill-sharing, local mentorship, and p2p tool lending. It was designed from the ground up featuring a polished "modern glass/premium" mobile app aesthetic, heavily relying on stark contrast, soft dynamic drop shadows, and high-fidelity typography.

## Features
- **Guild Feed**: Discovery home for workshops, tracking live active sessions, and premium custom dummy listings.
- **Expert Profile**: Detail view of community mentors, featuring vouch rates and legacy hours.
- **Skill Passport**: User profile showcasing vetted skills, verification progression, and sleek SVG honor badges.
- **Local Market**: Buy/sell up-cycled furniture and crafts locally with clear inline pricing and layout focus.
- **Tool Library**: Peer-to-peer neighborhood lending of power tools coupled with a minimal local map structure.
- **Live Session**: QR code handshake system with dummy validation to check into practical workshops securely.

## Tech Stack
- HTML5 Architecture
- Vanilla CSS (Glass/Modern UI Aesthetic)
- Vanilla Javascript (SPA navigation routing)
- Configured via Vite

## Development Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Preview Constraints
The prototype features dynamic structural layouts configured implicitly for mobile screens utilizing max-width containment (430px) allowing previews directly on larger screens through an emulated bezel structure.
